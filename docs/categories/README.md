# Widget Categories

Axios provides a comprehensive library of widgets. This guide serves as a quick reference map for locating exactly what you need.

---

## Core Containers

The foundation of any Axios UI.

- **[Windows](../widgets/window.md):** The primary container for all other UI elements.
- **[Formatting](../widgets/format.md):** Granular layout control using `SameLine`, `Indent`, and `Group`.
- **[Tables](../widgets/table.md):** Specialized containers for complex, multi-column grid layouts.

---

## Interactive Components

Standard elements for receiving and processing user input.

- **[Basic Widgets](../widgets/basic.md):** Buttons, checkboxes, and radio buttons.
- **[Input Boxes](../widgets/input-widgets.md):** Type-specific text fields for data entry.
- **[Draggable Inputs](../widgets/drag.md):** Numeric adjustment widgets via click-and-drag.
- **[Sliders](../widgets/slider.md):** Range-constrained selection sliders.
- **[Combo Boxes](../widgets/combo.md):** Dropdown selections and list picking.

---

## Content Display

Widgets for presenting information and structure.

- **[Text elements](../widgets/text.md):** Labels, multi-line text blocks, and formatted text.
- **[Images](../widgets/image.md):** Rendering Roblox assets, textures, and custom icons.
- **[Tree Nodes](../widgets/tree.md):** Collapsible headers and hierarchical data displays.
- **[Tabs](../widgets/tab.md):** Navigation bars and tabbed layout systems.
- **[Menus](../widgets/menu.md):** Desktop-style menu bars and interactive sub-menus.

---

## Visual Feedback

Widgets for telemetry and status indication.

- **[Plots & Graphs](../widgets/plot.md):** Progress bars, histograms, and data visualizations.
- **[Tooltips](../widgets/window.md#tooltip):** Context-sensitive floating help text.
